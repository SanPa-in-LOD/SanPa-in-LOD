Metadata Analysis

Metadata describe fundamental properties that are needed in order to gain knowledge about any kind of object. Thus an important task of this project was to provide a metadata standard for each item that was selected. In doing so we followed two general rules: if the metadata standard was explicitly stated by the provider and we believed that it was a safe choice, we kept that specific standard. If there were available standards more trustworthy we evaluated to get rid of the standard provided by the institution of origin and selected a more suitable standard. This is the case of the two items provided by Sebina Opac: from our understanding the provider uses UniMarc as a metadata standard, but we thought the ISBD would be a more appropriate choice, due to the fact that it is nowadays becoming more commonly used and it cover a more international area. The same could be said for the Ritratto of Vincenzo Muccioli, for which the host merely relies on the Open Graph Protocol, which is wide general standard for describing objects, while we believe that CCO is the most appropriate choice for museal entities. In all the other cases in which the provider did not specify any metadata standard, we opted for what we assumed was the safest choice: such is the case of Legge Vassalli-Jervolino, for which we used ELI (an European standard for legal items) and the photograph held by Regione Emilia-Romagna Assemblea Legislativa, that was described through IPTC.
