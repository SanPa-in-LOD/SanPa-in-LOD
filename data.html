Data Description

The description of data we have described our items on the basis of the conceptual model we have previously developed. Each item was described as a set of triples (subject/predicate/object) in natural language and it was put in tabular form in a .csv file. The following table is a visual representation of such .csv file, in which columns represent the part of the triple and rows are specific statements.
