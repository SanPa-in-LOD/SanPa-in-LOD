The project in brief

The project focuses on the history and the relative concepts related to the Community of San Patrignano. It has been describe through conceptual maps, E/R models, entities and 10 diffent items related to differnt cultural domains.The project has been organized in three different sections, which can be explored through the main navigation bar. One page is dedicated to the study of the domain, i.e. a description of the central concept of the porject, the motivations behind this choice and a description of the items. The knowledge organization part focuses on the different metadata standards that have been used to describe the aforementioned items. Finally, the knowledge representation part regards the descrpition of the concepts and entities that have been identified thourght RDF triples.
This web site is the main repository of the project, which can be also found on the GitHub page.
